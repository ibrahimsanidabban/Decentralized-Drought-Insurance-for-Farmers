# Decentralized Drought Insurance for Farmers

## Overview

This project implements a blockchain-based decentralized drought insurance system for farmers. The system utilizes smart contracts to automate the insurance process, from registration to claims processing, without requiring traditional intermediaries.

## Key Components

1. **Farm Registration Contract**: Digital registry for agricultural operations
2. **Weather Data Oracle Contract**: Provides verified climate information from trusted sources
3. **Soil Moisture Monitoring Contract**: Tracks drought conditions in specific farming areas
4. **Automated Payout Contract**: Triggers compensation based on predetermined conditions

## How It Works

1. Farmers register their farms, providing location and crop details
2. Weather and soil moisture data are continuously collected through oracle networks
3. Smart contracts analyze the data to detect drought conditions
4. Payouts are automatically triggered when predefined drought thresholds are met

## Benefits

- Eliminates paperwork and reduces administrative costs
- Provides transparent, tamper-proof record of all transactions
- Automates claims processing, delivering faster payments
- Removes the need for manual verification of drought conditions
- Increases financial inclusion for farmers in underserved regions
